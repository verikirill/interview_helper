# Interview Helper

Приложение для помощи на собеседованиях с AI-ассистентом.

## Возможности

- 🎤 Запись микрофона и системного звука одновременно
- 📸 Скриншоты экрана с анализом AI
- 👻 Сквозной режим (click-through) - окно не мешает кликам
- 🔒 Скрытие/показ окна по горячим клавишам
- 🎚️ Регулировка прозрачности
- ⚙️ Настраиваемые горячие клавиши
- 💾 Сохранение настроек

## Установка

```bash
cd interview-helper
npm install
npm start
```

## Горячие клавиши (по умолчанию)

| Действие | Комбинация |
|----------|------------|
| Скрыть/Показать | Ctrl+Shift+H |
| Сквозной режим | Ctrl+Shift+T |
| Скриншот | Ctrl+Shift+S |
| Спросить | Ctrl+Enter |
| Увеличить прозрачность | Ctrl+Shift+↑ |
| Уменьшить прозрачность | Ctrl+Shift+↓ |

## Настройка API

В настройках можно изменить:
- API Key (NVIDIA)
- API URL
- Модель

## Сборка

```bash
npm run build
```
